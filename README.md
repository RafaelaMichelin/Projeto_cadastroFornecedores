<h1 align="center"> Teste - Cadastro de Fornecedores </h1> 
<h2> :pushpin: Descrição do projeto:</h2>
<p>Esse projeto é uma aplicação web completa(CRUD) para o registro e gestão de fornecedores.</p>

<h2> :hammer_and_pick: Funcionalidades</h2>

- Cadastro(Create): Registo de fornecedores com campos para dados fiscais.
- Listagem(Read): Visualização de fornecedores numa tabela estilizada, com ordenação crescente.
- Exclusão (Delete): Remoção de registos com alerta de confirmação em JavaScript para evitar erros do utilizador.

<h2> :space_invader: Tecnologias utilizadas:</h2>

 - Backend: PHP 7.4 (utilizando a extensão **MySQLi**)
 - Banco de Dados: MySQL
 - Frontend: HTML5, CSS3, Bootstrap 3
 - Git/GitHub
 - Scripts/Bibliotecas: Jquery
 
   **jQuery:** Manipulação do DOM.
   
   **jQuery Mask:** Máscaras dinâmicas para CNPJ, CEP e Telefone.
   
   **jQuery Validate:** Validação do formulário de cadastro.
